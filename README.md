# riceleafdisease_classifier

I would like to present my portfolio project based on capstone project in Dicoding bootcamp, a CNN-based classification model to classify a rice leaf diseases image 

Data: Rice leaf disease dataset collected from several publishers on Kaggle The dataset contains 9 labels (1 healty leaf and 8 rice diseases)


Data sources :
https://www.kaggle.com/datasets/loki4514/rice-leaf-diseases-detection
https://www.kaggle.com/datasets/minhhuy2810/rice-diseases-image-dataset
https://www.kaggle.com/datasets/nirmalsankalana/rice-leaf-disease-image
https://www.kaggle.com/datasets/vbookshelf/rice-leaf-diseases

I've already aggregrated the datasets: https://drive.google.com/drive/folders/1paca0i8wcqNaRiPyyQszeQJruoS3At9c?usp=sharing


The model is also export to tflite format for deployment in Flutter-based android mobile app
